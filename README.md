<b>Prepare</b>
```
1. Login Mac AppStore (using brew mas)
2. xcode-select --install
3. chmod +x ./install.sh
```

<b>Install</b>
```
git clone https://github.com/segunlee/macOsPreset.git
cd ./macOsPreset
./install.sh

zsh + ohmyzsh + plugins only
./install.sh zsh
```
