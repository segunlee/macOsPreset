<b>Prepare</b>
```
1. Login Mac AppStore
2. xcode-select --install (Optional)
3. chmod +x ./install.sh
```

<b>Install</b>
```
git clone https://github.com/segunlee/macOsPreset.git
cd ./macOsPreset
sh ./install.sh
```
